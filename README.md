# X おすすめ非表示

iPhone/iPadのSafariで、Xの「おすすめ」タブを隠して「フォロー中」を表示するUserscripts用スクリプトです。

## インストール

1. iPhoneのSafariで次のURLを開きます。  
   https://raw.githubusercontent.com/aka525/x-following-userscript/main/hide-x-for-you.user.js
2. Safariの拡張機能メニューから「Userscripts」を開きます。
3. 表示されるインストール案内を押します。
4. SafariでXを再読み込みします。

## 動作

- Xのホーム画面で「おすすめ」タブを非表示にします。
- 「おすすめ」が選ばれている場合は「フォロー中」へ切り替えます。
- 日本語表示と英語表示に対応しています。
- 外部通信、データ収集、パスワードや投稿内容の保存は行いません。

## 無効化

Userscriptsの一覧で「X おすすめ非表示」をオフにして、Xを再読み込みしてください。

X側の画面構造が変更されると、将来動作しなくなる可能性があります。
