# X おすすめ・広告非表示

iPhone/iPadのSafariで、Xの「おすすめ」タブとタイムライン広告を隠して「フォロー中」を表示するUserscripts用スクリプトです。

## インストール・更新

1. iPhoneのSafariで次のURLを開きます。  
   https://raw.githubusercontent.com/aka525/x-following-userscript/main/hide-x-for-you.user.js
2. Safariの拡張機能メニューから「Userscripts」を開きます。
3. 表示されるインストールまたは更新の案内を押します。
4. SafariでXを再読み込みします。

## 動作

- Xのホーム画面で「おすすめ」タブを非表示にします。
- 「おすすめ」が選ばれている場合は「フォロー中」へ切り替えます。
- タイムラインに挿入される広告投稿を非表示にします。
- 日本語表示と英語表示に対応しています。
- 外部通信、データ収集、パスワードや投稿内容の保存は行いません。

広告は画面上で非表示にする方式です。広告データの読み込み自体を止めるものではありません。

## 無効化

Userscriptsの一覧で「X おすすめ・広告非表示」をオフにして、Xを再読み込みしてください。

X側の画面構造が変更されると、将来動作しなくなる可能性があります。
